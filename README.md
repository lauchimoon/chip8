# chip8
A robust chip8 interpreter. Made to start learning about emulators.

Uses [raylib](https://raylib.com) for displaying video

## Quick start
```sh
$ git clone https://github.com/lauchimoon/chip8.git
$ cd chip8/
$ PATH_TO_RAYLIB=path/to/raylib make
$ ./c8 <rom>
```
